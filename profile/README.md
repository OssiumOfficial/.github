# ossium

**Ocean of Open Source — Discover the best GitHub repositories blazingly fast.**

ossium is a free and open-source tool to explore trending and curated open source projects across GitHub. Built to help developers quickly find and filter repositories by language, popularity, and more — all without complex GitHub queries or bloated UIs.  
Made for speed. Free for everyone. Open for contributors.
